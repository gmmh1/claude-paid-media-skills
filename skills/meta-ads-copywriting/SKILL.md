# Meta Ads Copywriting

## Purpose
Write scroll-stopping, conversion-focused Meta ad copy — primary text, headline, description, CTA — that works with the visual creative rather than competing with it for attention.

## Trigger Conditions
- "Write Facebook/Instagram ad copy for [product]"
- "Write primary text and headlines for my Meta campaign"
- "Improve my Meta ad copy"
- "Give me copy variants for testing"

## Required Inputs
- Product/service and core offer
- Audience/awareness stage (cold prospecting vs. warm retargeting needs very different copy)
- Creative format the copy will run alongside (single image, video, carousel, Reels) since copy length/style should match the format's pacing
- Landing page or on-platform destination

## Core Capabilities

### Primary Text
- First 1-2 lines carry the weight — Meta truncates longer text behind "See more," so the hook must land before the cut
- Native, conversational tone outperforms formal ad-speak — Meta's format sits inside a social feed, and copy that reads like an ad gets scrolled past
- Length calibration by funnel stage: cold audiences often need more context/story (longer primary text can work if the hook is strong), warm retargeting audiences need less persuasion and more direct offer/CTA

### Headlines & Descriptions
- Headline (appears below the image/video on most placements): short, benefit- or offer-led, complements rather than repeats the primary text's hook
- Description (Feed placement, less prominent): supporting detail, often price/offer reinforcement

### Format-Aware Copy
- Single image/video: room for a fuller hook-to-CTA arc in primary text
- Carousel: each card needs its own micro-headline that works both standalone and as part of the sequence
- Reels/Stories: minimal on-screen text since the format is vision-led; primary text plays a smaller role than the video's own hook (coordinate with `meta-ads-video-reels`)

### Copywriting Frameworks
- PAS (Problem-Agitate-Solve) and Before/After work well in longer-form primary text for cold audiences discovering a problem they have
- Direct offer-led copy (price, promotion, urgency) for warm/retargeting audiences who already know the brand
- Social proof integration (review snippets, numbers) as a trust-building primary text opener for skeptical cold audiences

### CTA Selection
- Matching the CTA button to the actual destination and funnel stage (Shop Now vs. Learn More vs. Send Message vs. Sign Up) — mismatched CTAs (e.g., "Shop Now" leading to a blog post) hurt both performance and platform trust signals

## Workflow
1. Confirm offer, audience/awareness stage, and creative format.
2. Draft primary text with the hook in the first 1-2 lines, length calibrated to funnel stage.
3. Draft headline and description as complements, not repeats, of the primary text.
4. Match the CTA button to the actual destination.
5. Produce 2-3 distinct angle variants (not near-duplicate rewrites) for testing.
6. Flag any claim needing substantiation for policy review.

## Outputs
- Primary text (labeled by angle)
- Headline and description
- CTA recommendation
- Notes on format-fit (which variant suits which creative format)

## Rules
- Always front-load the hook in primary text — content after the "See more" truncation point gets a fraction of the readership.
- Match copy tone to platform-native, conversational register rather than formal ad copy — this is a stronger lever on Meta than on Search.
- Never mismatch CTA button to destination.
- Flag unsubstantiated claims (health, financial, "#1," guaranteed results) before finalizing.

- **Verification gate:** before this skill's output is used to spend real money, submit a compliance-sensitive claim, or go to a client as final, verify the mechanical specifics (limits, thresholds, policy rules) against current platform documentation and get explicit human sign-off — do not treat this skill's output as launch-ready without that check.
